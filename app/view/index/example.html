简单赋值调用 {literal} 模板输出 {$name} {/literal}
{$name}
数组赋值调用 {literal} 模板输出 {$data.name}/{$data.name2} {/literal}
{$data.name}/{$data.name2}
二维数组循环 两种方式 foreach | volist
foreach 模板使用
{literal} {foreach $list as $key=>$vo }
{$vo.name}
{/foreach}
{/literal}
foreach 的 key 从 0 起始
{foreach $list as $key=>$vo } {$key} >> {$vo.name}
{/foreach}
volist 模板使用
{literal} {volist name='list' id='vo' k='i' }
{$vo.name}
{/volist}
{/literal}
volist 的 key 从 1 起始
{volist name='list' id='vo' k='i' } {$i} >> {$vo.name}
{/volist}
使用函数 {literal} {:函数名} {/literal}
使用 MD5() 函数加密 $data.name
如 {literal} {:MD5($data.name)} / {$data.name | md5} {/literal}
加密后 {:MD5($data.name)} / {$data.name | md5}
默认值 default
{literal} {$data.names|default="没有字符串的时候出现我"} {/literal}
{$data.names|default="没有字符串的时候出现我"}
